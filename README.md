# Cloud Kitchen App Design

A UI/UX redesign of a cloud kitchen mobile application focused on simplifying the food ordering process through a clean layout, intuitive navigation, and a mobile-first approach.

##  About the Design
This project addresses common usability issues in food delivery apps, such as cluttered interfaces and complex navigation. The redesign emphasizes white space, visual hierarchy, and clarity to create a faster and more focused user experience.
